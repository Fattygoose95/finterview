# finterview 质量检查清单

本文档包含部署前、部署后和定期维护的质量检查项，确保网站功能完整、性能良好、兼容性达标。

## 部署前检查

### 1. 代码质量
- [ ] 所有HTML文件通过 `deploy-check.js` 检查
- [ ] 没有控制台错误（使用浏览器开发者工具检查）
- [ ] CSS无未使用的样式（使用CSS覆盖率工具）
- [ ] JavaScript无语法错误（使用 `node -c` 检查关键文件）

### 2. 功能完整性
- [ ] 首页行业选择界面正常显示
- [ ] 行业卡片可点击并跳转到对应题库
- [ ] 搜索功能正常工作（如有）
- [ ] 导航菜单链接正确
- [ ] 行业题库页面加载所有题目
- [ ] 筛选功能（星级、类型、难度）正常工作
- [ ] 清除筛选按钮重置所有筛选条件
- [ ] 高频题目列表页面正常显示
- [ ] 随机练习页面正常显示并可切换题目
- [ ] 移动端响应式布局正常

### 3. 性能检查
- [ ] 首页加载时间 < 3秒（模拟3G网络）
- [ ] 关键资源（CSS、JS）压缩
- [ ] 图片优化（如有）
- [ ] 启用浏览器缓存（通过 `netlify.toml` 配置）

### 4. 安全与合规
- [ ] 无敏感信息泄露（API密钥、密码等）
- [ ] HTTPS强制启用（Netlify默认）
- [ ] 安全头配置正确（X-Frame-Options, X-Content-Type-Options等）
- [ ] 无混合内容警告（所有资源使用HTTPS）

### 5. 兼容性
- [ ] Chrome最新版测试通过
- [ ] Firefox最新版测试通过
- [ ] Safari最新版测试通过
- [ ] Edge最新版测试通过
- [ ] 移动端Safari（iOS）测试通过
- [ ] 移动端Chrome（Android）测试通过
- [ ] 视口配置正确，支持移动设备缩放

## 部署后验证

### 1. 基本可访问性
- [ ] 生产URL可访问（无404错误）
- [ ] 所有页面返回HTTP 200状态码
- [ ] CSS和JS资源正确加载
- [ ] 控制台无资源加载错误

### 2. 功能验证
- [ ] 运行 `node scripts/verify-deployment.js remote --functional`
- [ ] 功能测试所有用例通过
- [ ] 跨页面导航正常工作
- [ ] 筛选功能在真实环境中有效

### 3. 性能验证
- [ ] 生产环境页面加载时间 < 2秒（有线网络）
- [ ] Lighthouse性能评分 > 90
- [ ] 首次内容绘制（FCP） < 1.5秒
- [ ] 最大内容绘制（LCP） < 2.5秒

### 4. SEO与元数据
- [ ] 所有页面有唯一且有意义的 `<title>`
- [ ] 所有页面有 `<meta name="description">`
- [ ] 结构化数据（如有）正确实现
- [ ]  robots.txt 允许搜索引擎索引

## 定期维护检查（每月）

### 1. 依赖更新
- [ ] 检查外部CDN资源（Font Awesome、Google Fonts）可用性
- [ ] 更新Netlify插件（如有）
- [ ] 检查Netlify构建环境更新

### 2. 内容更新
- [ ] 更新 `questions.js` 中的题目数据
- [ ] 检查所有链接是否仍然有效
- [ ] 更新文档以反映最新更改

### 3. 性能监控
- [ ] 使用Netlify Analytics或类似工具监控流量
- [ ] 检查页面加载时间趋势
- [ ] 监控错误率（404、500等）

### 4. 安全扫描
- [ ] 运行安全扫描工具（如Snyk、OWASP ZAP）
- [ ] 检查第三方资源的安全性
- [ ] 更新安全头配置（如有新标准）

## 紧急回滚检查

如果部署后发现问题，执行以下检查以回滚到上一个稳定版本：

- [ ] Netlify回滚到上一个成功部署
- [ ] 验证回滚后网站功能正常
- [ ] 检查数据一致性（题目数据等）
- [ ] 通知用户（如有必要）

## 测试脚本使用指南

### 部署前检查
```bash
node scripts/deploy-check.js
```

### 本地功能测试
```bash
node scripts/verify-deployment.js local --functional
```

### 远程部署验证
```bash
node scripts/verify-deployment.js remote --functional
```

### 仅功能测试
```bash
node scripts/functional-test.js [local|remote]
```

## 问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 页面空白 | JS错误、资源加载失败 | 检查控制台错误，验证资源路径 |
| 筛选功能无效 | JavaScript错误、数据格式问题 | 检查 `filters.js` 和控制台日志 |
| 移动端布局错乱 | 响应式CSS问题 | 检查视口设置和媒体查询 |
| 加载缓慢 | 资源过大、未启用缓存 | 压缩资源，检查缓存头配置 |
| 跨浏览器问题 | 浏览器特定兼容性问题 | 使用Polyfill或条件CSS |

- **SEO**: Lighthouse SEO评分 ≥ 90
